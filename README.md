# Personal Portfolio Website

A responsive, modern portfolio website built with HTML, CSS, JavaScript, and Bootstrap 5.

## Features

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Modern UI**: Clean design with custom branding colors
- **Interactive Elements**: Project filtering, modals, contact form validation
- **Performance Optimized**: Lazy loading images, optimized assets
- **Accessibility**: Semantic HTML, ARIA labels, keyboard navigation
- **SEO Friendly**: Proper meta tags, structured data

## Branding

- **Primary Color**: `#2ECC71` (Green)
- **Secondary Color**: `#FF8C00` (Orange)
- **Fonts**: Poppins (headings) and Inter (body)
- **Rounded Corners**: 8px border-radius
- **Shadows**: Soft shadows with hover effects
